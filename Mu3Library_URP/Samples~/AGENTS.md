# URP Samples Rules

- This folder owns URP package samples and demonstration assets only.
- Preserve sample import integrity, `.meta` files, and URP scene/material wiring.
- Prefer focused sample updates that validate or demonstrate URP package behavior without widening scope into base-package refactors.
- If a sample issue actually originates in runtime or shader code, make that package area the primary owner and keep the sample edit minimal.
- Shared wiki routes: [packages](../../docs/ai-agents/packages/README.md), [plans](../../docs/ai-agents/plans/README.md), [unity-yaml-guide](../../docs/ai-agents/guides/unity-yaml-guide.md).
